\abstract{Cosmomathica is a package for Mathematica by Wolfram Research,
Inc., providing a set of tools for computations in cosmology. Besides an
implementation of all the standard functions, such as comoving distance,
growth function, Hubble parameter, etc. it offers access to the 3rd party
software packages CAMB, Halofit+, Cosmic Emulator and a transfer function
fitting formula via a MathLink interface.}
